# Pulling this folder
  1. `git init`
  2. `git remote add origin https://github.com/YvesJudeMatta/learning`
  3. `git config core.sparsecheckout true`
  4. `echo "android/plain-ol-notes/*" >> .git/info/sparse-checkout`
  5. `git pull origin master`
